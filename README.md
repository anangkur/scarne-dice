## About The Project

This project is a result of a challenge from my office mate to solving a problem related to `data structure`. We've got the training source from [Applied CS Skills | Google for education](https://appliedcsskills.withgoogle.com/).

In this challenge I try to build a simple game using computer as the challenger to rolling the dice and once the scores reach 100 then the game will stop and announce the winner. This challenge focused on handling `while` and using `timer`.

For what I have learn, `while` is a method for looping with a condition. once the condition is fulfill the loop will stop.

`timer` is like a `while` but we can delay it's method so user can see what the computer do.
